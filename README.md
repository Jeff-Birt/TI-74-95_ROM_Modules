# TI-74/95_ROM_Modules

ROM modules for TI-74 and TI-95 which have been dumped. Typically each ROMs folder will have a notes.txt file which specific notations for that ROM. All documentation on hand is also included.

File extesnions:
.BIN  -- Binary dump of ROM, may be read directly from ROM
.JPG  -- Image :)
.PDF  -- Documentation in PDF format
.PNG  -- Image :)
